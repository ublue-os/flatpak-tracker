# About Flatpak Runtime Tracker

This site tracks runtime updates for flatpak applications shipped by Universal Blue (Bluefin, Aurora, Bazzite).

## About

The Flatpak Runtime Tracker automatically monitors flatpak packages for outdated runtimes and creates GitHub issues to help maintainers update their applications.

### Features

- **Automated Monitoring**: Weekly checks for runtime updates
- **Multi-Source Tracking**: Monitors packages from Bluefin, Aurora, and Bazzite
- **Popular App Prioritization**: Highlights high-impact applications
- **Historical Tracking**: Complete changelog of all updates

### How to Help

1. **Test existing PRs**: Many apps already have updates pending on Flathub
2. **Submit updates**: Follow the instructions in GitHub issues
3. **Report issues**: Help maintainers by testing updated applications
